Process

How it works

Three steps from source to certified. Fully automated, fully auditable.

1
🔍

Scan

Your agent package is automatically scanned against the OWASP LLM Top 10. A score below 70 is refused — no exceptions.

score ≥ 70 required
2
✅

Sign

A unique License ID is generated and a cryptographic hash of your package is registered on the agentverif.com registry. A SIGNATURE.json is injected.

AV-XXXX-XXXX
3
🔐

Certify

Buyers verify the certificate at verify.agentverif.com or via the CLI. Tampered packages are flagged instantly.

sha256 hash-locked

CLI quickstart

As a vendor

agentverif-sign sign ./agent.zip # ✅ Signed successfully # License: AV-84F2-91AB # Tier: indie

As a buyer

agentverif-sign verify ./agent.zip # ✅ UNREGISTERED # Verify: verify.agentverif.com/…

Output is JSON-compatible for CI/CD pipelines, LLM tool calls, and MCP integrations (agentverif-sign verify --json).
