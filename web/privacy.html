Privacy Policy

Last updated: April 13, 2026

Overview

agentverif is a read-only agent verification service. When you or an AI assistant submits a license ID, we look it up in our public registry and return the result. We do not create accounts, track users, or store any data about the people who use this service.

This policy covers agentverif.com, verify.agentverif.com, api.agentverif.com, and mcp.agentverif.com.

Data we collect

Data Purpose Retention
license_id submitted for verification Registry lookup — returned immediately, not stored per-user Up to 30 days in server access logs, then auto-deleted
Anonymous request logs (IP address, timestamp, endpoint) Rate limiting and abuse prevention 30 days maximum, then auto-deleted

License verification results are never stored per-user. We return the registry record to you and immediately discard the transaction.

Data we do not collect

  • No conversation history from Claude, ChatGPT, or any LLM
  • No user identity, name, email, or authentication data
  • No cookies or browser fingerprinting
  • No third-party analytics (no Google Analytics, Mixpanel, etc.)
  • No advertising identifiers
  • No payment information
  • No agent ZIP file contents — only the license ID you explicitly provide

MCP plugin — Claude integration

When agentverif is used as an MCP server connected to Claude or another AI assistant, the following applies:

  • Only the license_id parameter is transmitted to agentverif servers — nothing else from your conversation
  • No conversation context, messages, or history is ever sent to or stored by agentverif
  • Anthropic does not share user data with agentverif
  • The MCP server endpoint is stateless — no session data is retained between requests

The MCP server returns only what is already in the public registry. There is no mechanism for agentverif to read, store, or influence your AI conversation.

Data retention

  • Server access logs: 30 days maximum, then permanently deleted
  • License verification results: not stored per-user — returned and discarded immediately
  • License registry records: stored until revoked or expired by the issuer

Data sharing

We do not sell, rent, or share personal data with third parties. License IDs submitted for verification are forwarded to an internal registry API on the same server. No external third-party services receive verification requests.

Cookies and tracking

agentverif sets no cookies and uses no tracking technologies. There are no analytics scripts, advertising pixels, or session identifiers on any agentverif page.

Security

All traffic between users and agentverif is encrypted via HTTPS (TLS 1.2+). Internal API calls are made over localhost only and never leave the server.

Your rights

Because agentverif does not collect personally identifiable information, standard data subject requests (access, correction, deletion) are generally not applicable. If you have concerns about data processed in connection with your use of this service, contact us at the address below.

Changes to this policy

We may update this policy when our practices change. Material changes will be announced at agentverif.com. The "Last updated" date at the top of this page reflects the most recent revision.

Contact

Questions or concerns? Reach us at hi@agentverif.com.
